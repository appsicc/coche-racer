# Coche Racer

Juego web sencillo de esquivar coches. Funciona en ordenador y movil.

## Jugar

Abre `index.html` en el navegador o publica el repositorio con GitHub Pages.

Controles:

- Flechas izquierda y derecha, o teclas `A` y `D`
- Botones en pantalla en movil
- `P` o espacio para pausar

## Publicar con GitHub Pages

1. Entra en Settings.
2. Abre Pages.
3. En Source, elige la rama `main` y la carpeta raiz.
4. Guarda los cambios.
